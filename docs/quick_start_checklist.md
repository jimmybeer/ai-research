# Quick Start Checklist

## 1. Initial Setup
- [ ] Clone the repository
- [ ] Read `README.md`
- [ ] Verify directory structure (`docs/`, `lessons/`, `cheatsheets/`, `prompts/`, `agents/`, `scripts/`, `references/`, `workflows/`)
- [ ] Review `docs/project_overview.md`
- [ ] Skim `docs/contributing.md`

## 2. Environment Verification
- [ ] ChatGPT web access confirmed
- [ ] IDE (VS Code) with Codex integration installed and authenticated
- [ ] Codex CLI installed (`pip install codex-cli`) and configured with API key
- [ ] Test Codex in each interface (web prompt, IDE chat, CLI command)

## 3. First Day Tasks
- [ ] Complete `docs/getting_started_walkthrough.md`
- [ ] Exercise 1: Run a prompt in the web interface
- [ ] Exercise 2: Invoke an agent in IDE chat
- [ ] Exercise 3: Use CLI with a repo resource and log the session
- [ ] Add your first note/prompt/reference to the repo

## 4. Week One Goals
- [ ] Read `docs/codex_interfaces_guide.md`
- [ ] Read `docs/agent_system_guide.md`
- [ ] Complete `lessons/using_codex_cli_basics.md`
- [ ] Run all four agents at least once
- [ ] Create 1â€“2 custom prompts or adapt existing ones
- [ ] Execute the workflow in `workflows/creating_a_new_lesson.md`

## 5. Ongoing Habits
- [ ] Capture learnings weekly in `references/session_logs/`
- [ ] Review prompts before major work sessions
- [ ] Consult agents for complex or repetitive tasks
- [ ] Update cheatsheets when you adopt new commands or flags
- [ ] Follow the maintenance schedule in `docs/maintenance_guide.md`
